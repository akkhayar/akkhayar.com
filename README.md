# akkhayar.com

သည်ပရောဂျက်တွင် [`pnpm`](https://pnpm.io/) ကိုအသုံးပြုထားသည်။ မရှိသေးလျှင် [သည်အညွှန်း](https://pnpm.io/installation) ကိုဖတ်ပြီးထည့်သွင်းပါ။
 `npm` သို့ `yarn` သုံးလို့မရပါ။

## Quickstart

Dependencies များကို ထည့်သွင်းပါ။

```bash
pnpm i
```

Server စလို့ရပါပြီ။

```bash
pnpm run dev
```
